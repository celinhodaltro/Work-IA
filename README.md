# AI Office OS

Plataforma inteligente de automação de escritório com arquitetura orientada a eventos.

## Repositório
- **GitHub:** https://github.com/celinhodaltro/Work-IA.git
- **Documentação:** `docs/decisions/`

## Pré-requisitos
- .NET 8 SDK
- Docker Desktop
- Git

## Build

```bash
dotnet build
```

## Testes

```bash
dotnet test
```

## Infraestrutura

```bash
docker compose up -d
```
